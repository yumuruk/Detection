# Detection
Ultralytics/MMdetection detector

To use mmdetection you need to follow these steps


```
git clone https://github.com/yumuruk/Detection.git

cd Detection

git clone https://github.com/open-mmlab/mmdetection.git

cd mmdetection

python setup.py develop # You should activate virtual environments

cd ..

```


To use ultralytics

```
pip install ultralytics

python train_yolo.py

```
